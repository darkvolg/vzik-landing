🖥 Windows — защита через «Пользовательские правила»

На Windows всё проще чем на Android: российские сайты уже идут напрямую через наш RU-сервер (Yandex Cloud) — банки видят обычного российского пользователя. Для дополнительной гарантии добавим домены банков и Госуслуг в «Прямое» внутри v2RayTun.

✅ Хорошая новость: на Windows по умолчанию стоит режим «Системный прокси» (видно в Настройки → Настройки трафика → Режим). В этом режиме нативные Windows-программы (не браузеры) и так идут мимо VPN. Банковские клиенты для ПК не подвержены методичке — она их просто не видит.

Anubis — автоматическая защита (для продвинутых)

Anubis — это open-source приложение, которое автоматически замораживает приложения-шпионы (Сбер, Ozon, Госуслуги) когда у вас включён VPN. Более надёжно чем split-tunneling, потому что замороженное приложение физически не существует в системе — оно не может ничего проверить.

🧊 Как работает

Anubis использует Shizuku API чтобы давать команды системе без root-прав. Вы распределяете приложения по 3 группам:

  • «Без VPN» — замораживаются при включении VPN (банки, Госуслуги, маркетплейсы)
  • «Только VPN» — замораживаются при выключении VPN (Instagram, YouTube)
  • «Запуск с VPN» — автоматически активируют VPN при запуске
📥 Установка
1
Установите Shizuku из Google Play
Shizuku — это системная утилита которая даёт приложениям «безопасные» root-команды. Бесплатно, безопасно, в Google Play.
2
Запустите Shizuku
Следуйте инструкции внутри приложения — активация через ADB или беспроводную отладку. Это делается один раз.
3
Установите Anubis с GitHub
Ищите по запросу «Anubis Android VPN freeze». Скачайте APK с официального репозитория, установите.
4
Добавьте Сбер, Ozon, Госуслуги в группу «Без VPN»
Также добавьте все банки, Wildberries, RUTUBE, MAX. Теперь при включении VPN они автоматически заморозятся и не смогут вас задетектить.

Почему iPhone — самый безопасный вариант

Минцифры в своей же методичке официально признаёт:

«На iOS выявление VPN существенно ограничено из-за политики изоляции приложений.»

🔒 Почему iOS безопаснее
  • Apple не даёт приложениям «видеть» другие приложения (sandbox)
  • Нет системного API для проверки VPN (в отличие от Android ConnectivityManager)
  • Сбер не может узнать у системы активен ли VPN
  • Банки и Госуслуги не могут прочитать сетевые интерфейсы tun0/ppp0
